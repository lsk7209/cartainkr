# 자동차 유지비 계산기 사용 가이드

URL: https://cartain.kr/calculator

## 개요

카테인 자동차 유지비 계산기는 차량 할부금·연료비·보험료·자동차세를 한 번에 계산해 월 유지비와 연간 총비용을 산출합니다. 무료로 이용 가능하며 별도 회원가입이 필요 없습니다.

## 사용 방법

1. **차종 선택** — 경차·소형·중형·대형·SUV·전기차 중 선택하거나 차량 가격 직접 입력
2. **할부 조건 입력** — 할부 기간(개월), 금리(%)
3. **연비 정보 입력** — 연비(km/L), 월 주행거리(km), 유가(원/L)
4. **보험료 입력** — 월 자동차 보험료(원)
5. **결과 확인** — 항목별 비용 + 월 총유지비 + 연간 총비용

## 계산 공식

### 할부금 (원리금균등상환)
```
월 할부금 = P × [r(1+r)^n] / [(1+r)^n - 1]
P = 차량 가격(원), r = 월 이율(연 금리 ÷ 12), n = 할부 개월 수
```

예시: 3,000만 원, 연 5%, 60개월 → 월 약 56.6만 원

### 월 연료비
```
월 연료비 = (월 주행거리 ÷ 연비) × 유가
```

예시: 1,500km ÷ 12km/L × 1,700원/L = 약 21.3만 원

### 월 자동차세 (환산)
```
연간 자동차세 = 배기량(cc) × 세율
월 환산 = 연간 자동차세 ÷ 12
```

## 자주 묻는 질문

### 계산 결과가 실제 비용과 다를 수 있나요?
네, 보험료는 운전 경력·사고 이력에 따라 크게 다르고 유가는 매일 변동됩니다. 계산기는 입력값 기반 추정치를 제공합니다.

### 전기차 충전비는 어떻게 계산하나요?
전기차는 연비 대신 전비(km/kWh)와 충전 단가(원/kWh)로 계산합니다. 가정용 완속 충전 기준 약 100원/kWh, 공공 급속 충전 기준 약 250~350원/kWh.
